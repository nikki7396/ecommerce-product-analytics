# ecommerce-product-analytics
End-to-end analysis of e-commerce user behavior including funnel analysis, category conversion rates, cart abandonment insights, and purchase time patterns.

## 📊 Project Overview

This project analyzes user behavior on an e-commerce platform using a dataset containing over 3 million user interactions.  

The goal is to understand how users interact with products and identify opportunities to improve **conversion rates, reduce cart abandonment, and optimize purchase timing**.

## Business Questions

This analysis focuses on answering the following questions:

1. Where do users drop off in the purchase funnel?
2. Which product categories have the highest conversion rates?
3. How many users abandon items in their cart?
4. What time of day do most purchases occur?
5. What types of users interact with the platform?

## Analysis Performed

The following analyses were conducted:

 Data Cleaning and Preparation
 Funnel Analysis (View → Cart → Purchase)
 Category Performance Analysis
 Category Conversion Rate Analysis
 Cart Abandonment Analysis
 Purchase Time Analysis
 User Segmentation

## Key Insights

• A major drop occurs between **product views and cart additions**.  

• Certain product categories such as **stationery cartridges and apparel gloves show the highest conversion rates**.  

• Over **50% of cart users remove items before completing a purchase**, indicating significant cart abandonment behavior.  

• Most purchases occur between **9 AM and 1 PM**, with another peak during **evening hours**.

##  Business Recommendations

 Improve product pages to increase add-to-cart behavior.
 Highlight high-conversion product categories in promotions.
 Address cart abandonment with targeted discounts and reminders.
 Schedule marketing campaigns during peak purchase hours.
